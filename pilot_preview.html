The retrieval taster — pick holes in it

New shape, as you asked: the cue is the whole English sentence with the tense in (parentheses). The learner only fills the verb. Twelve questions below — the last four are the ones with a rival verb, where it gets weird.

The help is loud on every question. Below each blank sits a blue bar: “Don’t know the Italian for ‘to speak’? → parlare”. Nobody is stuck for not knowing a word — using it just quietly notes the vocabulary for review. That’s what makes this fair at A1.

The ones where a rival verb exists

Here the English could pull two ways (know, watch, leave, get up). The help names the exact verb, so it’s never really a guessing game. And if they still pick the other verb, that is a vocabulary note only — never a grammar mark of any kind. Grammar is scored solely on forming the verb we meant.

I know the right answer (a fact I hold now)
la risposta giusta.  →  so
?Don’t know the Italian for “to know (a fact)”?→ sapere
If they write “conosco”: that’s conoscere (know a person/place); the help names sapere. Logged as a vocabulary note — the grammar is untouched.
You watch too much television (in general)
Tu troppa televisione.  →  guardi
?Don’t know the Italian for “to watch”?→ guardare
If they write “vedi”: that’s vedere (to see); the help names guardare. Vocabulary note only — the grammar is untouched.
Every summer we leave for the seaside (a habit)
Ogni estate per il mare.  →  partiamo
?Don’t know the Italian for “to leave / depart”?→ partire
If they write “lasciamo”: that’s lasciare (leave something behind); the help names partire. Vocabulary note only — the grammar is untouched.
In the morning I get up very early — leave the bed (every day)
La mattina molto presto.  →  mi alzo
?Don’t know the Italian for “to get up”?→ alzarsi
If they write “mi sveglio”: that’s svegliarsi (wake up); the help names alzarsi. Recorded as a vocabulary note only — never a grammar mark. This is the case you flagged, now the rule for every wrong verb.
